## 🎨 Seções

1. **Hero** - Headline impactante com CTA e estatísticas
2. **Serviços** - Grid com 9 serviços especializados
3. **Diferenciais** - 6 motivos para escolher a oficina

## 🚀 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/tofariasti/oficinamecanica.git
```

2. Abra o arquivo `index.html` no navegador para ver a versão com preview responsivo

3. Ou abra `site/index.html` para ver a landing page em tela cheia

## 📞 Integração WhatsApp

Para personalizar o número do WhatsApp, edite o arquivo `site/assets/js/script.js`:

```javascript
const whatsappNumber = '5551999999999'; // Seu número aqui
```

A mensagem é pré-formatada com campos para:
- Veículo (marca/modelo)
- Serviço desejado
- Data preferencial
- Telefone para contato
